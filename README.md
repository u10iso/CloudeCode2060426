# シンプルアラームアプリ

シンプルで使いやすいアラームアプリです。ブラウザで動作します。

## 機能

- ⏰ 時刻を指定してアラームを設定
- 📱 複数のアラームを同時管理
- 🔔 アラーム時刻に音と通知で知らせます
- 💾 アラーム設定はブラウザに自動保存されます
- 📱 レスポンシブデザイン対応

## 使い方

1. `index.html` をブラウザで開きます
2. 「時刻を選択」から希望の時刻を選択します
3. 「アラームを設定」ボタンをクリックします
4. 設定したアラームが一覧に表示されます
5. アラーム時刻になると、音と通知で知らされます

## ファイル構成
